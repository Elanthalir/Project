# To run the project.
# Open command prompt in the file location and run the main file using the below command.
# streamlit run main.py
# and press enter a user interface will appear in your web browser.
# In this we are using streamlit for user interface.
# In the home page you can see information about liver disease.
# In data you can view your dataset information. and press on the dataset link bottom on the page. To download the dataset you need internet connection.
# In the prediction page you can enter the values of the liver medical reports data in the required filed and predit that the patient has liver disease or not.
# In the visualise page you can view graphical represention by mathplot.
